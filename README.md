# GST Calculator Web Application

A modern, responsive, and easy-to-use Goods and Services Tax (GST) Calculator built with React.js and CSS.

## Features

- **Base Amount Input**: Easily enter your base amount.
- **GST Rate Selection**: Choose from standard GST slabs (5%, 12%, 18%, 28%).
- **Instant Calculation**: Quickly calculates GST amount and total amount.
- **Form Validation**: Clean error handling to prevent incorrect entries.
- **Modern UI**: Attractive gradient header, smooth hover effects, and a responsive card layout.
- **Developer Attribution**: Displays custom developer info and a "Built for Digital Heroes" outbound link.

## Tech Stack

- **React**: Functional components and hooks (`useState`).
- **Vite**: Superfast frontend build tool.
- **Vanilla CSS**: Used for rich aesthetic styles, modern typography, and micro-animations.

## Getting Started

Follow these steps to run the application locally:

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the local server URL (usually `http://localhost:5173`).

## Deployment

This project is configured and ready for deployment on [Vercel](https://vercel.com).
To deploy:
1. Connect your GitHub repository to Vercel.
2. Vercel will automatically detect Vite and run `npm run build`.
3. The output directory is automatically set to `dist`.
